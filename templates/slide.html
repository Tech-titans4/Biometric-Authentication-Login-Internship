ABOUT

*

The proposed project is Template for Biometric Authentication Landing Page.

It implements Face ID authentication in a Django web application using the Face Recognition Library.

*

First the registered user has to take a photo using webcam provided at the login page.

Once the clicked photo is encoded it is compared with the already saved photo.

*

If it matches then the user is logged in or else the user is denied access.
